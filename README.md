# codePlatform

## 个人开发辅助平台

辅助平常开发工作的工具平台

## 主要支线

- **master 分支**（ 线上正式环境版本 ）

- **test 分支** 开发者版本（ 线上预发布环境版本，用于测试 ）

多环境发布过程统一由 Jenkins 发布
